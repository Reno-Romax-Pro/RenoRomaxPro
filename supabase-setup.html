☁️ Supabase Setup

Настройка облачной базы данных для вашего сайта

1 Создайте аккаунт Supabase

Supabase — это бесплатная PostgreSQL база данных в облаке.

Открыть Supabase.com
Бесплатный план включает:
  • 500 MB базы данных
  • 1 GB файлового хранилища
  • Безлимитные API запросы

2 Создайте новый проект

  1. Нажмите "New Project"
  2. Название проекта: reno-romax-pro
  3. Придумайте надёжный пароль для базы данных
  4. Выберите регион ближе к вам
  5. Нажмите "Create new project"
  6. Подождите ~2 минуты пока проект создастся

3 Получите API ключи

  1. Перейдите в Settings → API
  2. Скопируйте Project URL
  3. Скопируйте anon public ключ
Пример: https://abcdefgh.supabase.co
Этот ключ безопасен для использования в frontend коде

4 Создайте таблицы в базе данных

Выполните этот SQL в Supabase SQL Editor:

-- Таблица товаров
CREATE TABLE products (
  id SERIAL PRIMARY KEY,
  handle TEXT UNIQUE NOT NULL,
  title TEXT NOT NULL,
  description TEXT,
  price DECIMAL(10,2),
  image TEXT,
  category TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Таблица категорий
CREATE TABLE categories (
  id SERIAL PRIMARY KEY,
  name TEXT UNIQUE NOT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Включить Row Level Security (RLS)
ALTER TABLE products ENABLE ROW LEVEL SECURITY;
ALTER TABLE categories ENABLE ROW LEVEL SECURITY;

-- Разрешить публичное чтение
CREATE POLICY "Allow public read" ON products FOR SELECT USING (true);
CREATE POLICY "Allow public read" ON categories FOR SELECT USING (true);

-- Разрешить публичную запись
CREATE POLICY "Allow public insert" ON products FOR INSERT WITH CHECK (true);
CREATE POLICY "Allow public update" ON products FOR UPDATE USING (true);
CREATE POLICY "Allow public delete" ON products FOR DELETE USING (true);

CREATE POLICY "Allow public insert" ON categories FOR INSERT WITH CHECK (true);
CREATE POLICY "Allow public update" ON categories FOR UPDATE USING (true);
CREATE POLICY "Allow public delete" ON categories FOR DELETE USING (true);

-- Индексы для быстрого поиска
CREATE INDEX idx_products_category ON products(category);
CREATE INDEX idx_products_handle ON products(handle);
  1. Перейдите в SQL Editor в Supabase
  2. Нажмите "New query"
  3. Вставьте SQL код выше
  4. Нажмите "Run" (или Ctrl+Enter)

5 Скачайте файл конфигурации

Нажмите кнопку ниже, чтобы скачать готовый файл с вашими ключами:

После скачивания замените файл supabase-config.js в папке вашего проекта.

6 Загрузите данные в облако

После обновления конфигурации:

  1. Загрузите файлы на Netlify
  2. Откройте Admin Panel
  3. Перейдите в Settings
  4. Нажмите "🔄 Sync Local Data to Supabase"
